## Overview
Developed a fully functional web app for Grab's merchant-partners (MEX), enabling login and interaction with a chat-based AI assistant for real-time business insights and personalized guidance.

## Key Features
- Real-time business insights through AI chat interface
- Sales summaries and inventory monitoring
- Customer behavior analysis
- Seasonal trends analysis
- Multilingual platform support

## Technical Implementation
- Developed a responsive web application
- Integrated AI/ML capabilities for business insights
- Created a multilingual interface
- Implemented secure authentication system

## Impact
- Enhanced merchant decision-making through data-driven insights
- Improved business operations through automated monitoring
- Personalized recommendations based on business type, size, and location
