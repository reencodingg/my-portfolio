## Project Overview
NBA Manager is a comprehensive Java application designed to streamline NBA team management operations. The system handles player management, contract negotiations, injury tracking, and travel route optimization.

## Key Features
- Player Management System
  - Track player statistics and performance
  - Manage player rosters and team composition
  - Handle player contracts and salary negotiations
- Injury Tracking System
  - Monitor player injuries and recovery timelines
  - Manage player availability for games
- Route Optimization
  - Calculate optimal travel routes between venues
  - Visualize team travel schedules
  - Minimize travel time and costs

## Technical Implementation
- Developed in Java with object-oriented design principles
- Implemented data structures for efficient player and team management
- Created a route optimization algorithm for travel planning
- Designed an intuitive user interface for easy navigation

## Impact
- Streamlined team management processes
- Improved decision-making through data visualization
- Enhanced travel planning efficiency
- Reduced administrative overhead
