## Overview
Designed and prototyped a mobile platform enabling users to give away or request food and household items within their community.

## Key Features
- Food and item sharing functionality
- Geolocation-based community matching
- Category-based filtering system
- Interactive request/offer flows
- User-friendly interface

## Technical Implementation
- Developed native Android application
- Implemented geolocation services
- Created intuitive UI/UX design
- Built secure user authentication
- Integrated real-time updates
- [View source code on GitHub](https://github.com/yanayusman/SharePlate)

## Impact
- Reduced food waste in communities
- Improved accessibility to resources
- Fostered community connections
- Enhanced user experience through intuitive design
