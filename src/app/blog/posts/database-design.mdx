# Database Design and Development Project

**COCO Furniture Marketing Sdn Bhd | Oracle APEX, SQL**  
*Oct 2024 - Dec 2024*

Developed a relational database system using Oracle APEX and SQL to manage inventory, sales, payments, and suppliers for a furniture company, based on simulated data.

Implemented business rules and created secure user interfaces for staff to handle transactions, commissions, and reporting tasks.

Collaborated with company representatives to ensure the system aligned with real-world workflows while maintaining data confidentiality.
